# Design Inspiration Sites

Need references for your landing page design? Browse these curated galleries for landing page and web design inspiration:

| Site | What you'll find |
|---|---|
| [landingfolio.com](https://www.landingfolio.com) | Landing page design inspiration — hero sections, pricing pages, CTAs |
| [godly.website](https://godly.website) | Web design inspiration, curated daily |
| [mobbin.com](https://mobbin.com) | Mobile & web app UI/UX screenshots (400k+) |
| [saasinterface.com](https://saasinterface.com) | SaaS UI/UX examples — dashboards, onboarding, settings pages |

Screenshot anything that catches your eye, name it descriptively, and drop it in the `references/` folder before Phase 2.
